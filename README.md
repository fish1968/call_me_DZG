1. [劫镖计算](劫镖计算)

   1. 🌝根据对方的全部挚友数据，以及战斗力推算对方的可能阵容
   2. 🌝根据推测的阵容，安排我方有$P_{pass}$以上通过率的最小战斗力和阵容
   3. 🌗根据对方的数据组，判断对方的组合是那些，大大降低可能的排列数。还需要完成对应的我方出战阵容生成。
   4. 🌑考虑出战的损耗值
   5. 🌑多几个输出，可以按照不同的角度获得不同的阵容（花元宝的数量、战斗力总和值，方差值）
